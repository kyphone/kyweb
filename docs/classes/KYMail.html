\KYMail

KYMailクラスはテンプレートからメールを作成、送信します。

メールのテンプレートは /private/mail フォルダに *.txt として保存します。

template()

template(string  $name) : object

メールのテンプレート名を指定します。

テンプレート名は拡張子 .txt なしで指定します。

テンプレートは /private/mail に保存します。

Parameters

string $name

テンプレート名

Returns

object —

自分自身(KYMailオブジェクト)

Examples

<?php
$mail->template("sample");
?>

send()

send(string  $from, string  $to, string  $cc = "") : object

テンプレートを読み込み、入れ替え部分を処理し、メールを送信します。

Parameters

string $from

送信元メールアドレス
string $to

送信先メールアドレス
string $cc

Ccメールアドレス

Returns

object —

自分自身(KYMailオブジェクト)

Examples

<?php
$mail->template("sample");
$mail->assign("NAME", "xxx");
$mail->send("from@sample.com", "to@sample.com", "cc@sample.com");
?>
